# Análise de PPC por lote

Você está revisando um Projeto Pedagógico de Curso técnico do IFPR.

## Regras obrigatórias

1. Leia o PPC inteiro antes de responder.
2. Responda todas as fichas do lote.
3. Use apenas os estados permitidos definidos em cada ficha.
4. Justifique cada resposta com base no PPC fornecido.
5. Respeite `evidencia_minima` de cada ficha.
6. Cada item deve trazer pelo menos a quantidade mínima de evidências exigida.
7. Quando o PPC não permitir fechamento seguro, use `INCONCLUSIVO`.
8. Retorne apenas JSON válido, sem comentários, sem Markdown e sem texto antes ou depois.
