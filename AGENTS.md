# Instruções para agentes

Ao fazer alterações em qualquer skill deste repositório, leia primeiro o `README.md` da própria skill. Use esse arquivo para entender o propósito, o fluxo de uso, os comandos de manutenção e as convenções locais antes de editar scripts, fichas, prompts, testes ou documentação.

Se a alteração envolver uma base gerada ou índice consolidado, procure no `README.md` da skill o comando de regeneração ou validação correspondente e execute-o antes de concluir.
