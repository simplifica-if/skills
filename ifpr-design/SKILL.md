# IFPR Design

Use esta skill para aplicar a identidade visual institucional do IFPR com consistência em apresentações, documentos, páginas, relatórios, interfaces e peças visuais.

## Regras Essenciais

- Use Open Sans como fonte institucional quando o contexto permitir.
- Use verde IFPR `#1F8A42`, vermelho IFPR `#C72E2E`, cinza escuro `#2D2D2D` e branco `#FFFFFF`.
- Para texto de corpo, prefira `#2D2D2D` sobre branco; o verde institucional sobre branco atende contraste apenas para texto grande ou negrito.
- Preserve proporção, cores e área de proteção do logo.
- Não distorça, rotacione, redesenhe, aplique sombra, remova elementos ou altere as cores do logo.
- Não use apenas cor para comunicar informação; mantenha contraste WCAG AA sempre que possível.

## Como Usar

1. Identifique o tipo de material: apresentação, documento, página, interface ou arte.
2. Consulte apenas as referências necessárias para a tarefa.
3. Aplique cores, tipografia, logo e contraste conforme o contexto.
4. Ao entregar HTML/CSS, reutilize `colors.css` ou os valores de `colors.json` quando for adequado.
5. Ao gerar documentos ou apresentações, use `assets/logo-ifpr.png` e preserve a área de proteção.
